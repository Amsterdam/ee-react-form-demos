# Handling Remote Data

Sometimes forms need to be prefilled with data loaded asynchronously (for example: editing an existing record, loading user details or restoring a draft submission).

This page shows the two common approaches:

- **Controlled React forms**, where you update state manually using `setState()`
- **React Hook Form**, where you preload values using `reset()`

The key detail is that remote data usually arrives *after* the initial render, so you must explicitly update the form values once the request completes.

**Pro tip** - Keep your form data object as flat and with as few child array/object values as possible to keep your components and state handling simple.

## Caveats when prefilling async form data

When loading initial form values from an API, be careful not to overwrite user input.

Common issues include:

- **Refetching triggers a reset loop** (for example if the `useEffect` dependencies are wrong).
- **Prefill overwrites user edits** if the user can start typing before the API call completes.
- **React Hook Form dirty state** can be confusing if you call `reset()` multiple times.

In general, only prefill once (or only when the record ID changes) and avoid calling `reset()` repeatedly unless you intentionally want to discard local edits.

## Plain HTML5 Form in React

The key step here is calling the `setFormData()` function after the `fetch` call.

```tsx
import { useEffect, useState, FormEvent } from "react";

type FormData = {
  name: string;
  email: string;
  message: string;
};

export default function ContactForm() {
  const [formData, setFormData] = useState<FormData>({
    name: "",
    email: "",
    message: "",
  });

  useEffect(() => {
    async function fetchInitialData() {
      // Simulated API call
      const res = await fetch("/api/interesting-data");
      const data: FormData = await res.json();
      setFormData(data);
    }
    fetchInitialData();
  }, []);

  const handleChange = (e: React.ChangeEvent<HTMLInputElement | HTMLTextAreaElement>) => {
    const { name, value } = e.target;
    setFormData((prev) => ({ ...prev, [name]: value }));
  };

  const handleSubmit = (e: FormEvent<HTMLFormElement>) => {
    e.preventDefault();
    console.log("Submitting", formData);
    // Send to API...
  };

  return (
    <form onSubmit={handleSubmit} noValidate>
      <input
        type="text"
        name="name"
        value={formData.name}
        onChange={handleChange}
        placeholder="Name"
      />
      <input
        type="email"
        name="email"
        value={formData.email}
        onChange={handleChange}
        placeholder="Email"
      />
      <textarea
        name="message"
        value={formData.message}
        onChange={handleChange}
        placeholder="Message"
      />
      <button type="submit">Submit</button>
    </form>
  );
}
```

## React Hook Form

The key step here is calling the `reset()` function after the `fetch` call.

```tsx
import { useEffect } from "react";
import { useForm, Controller } from "react-hook-form";

type FormData = {
  name: string;
  email: string;
  message: string;
};

export default function ContactFormControlledRHF() {
  const { control, handleSubmit, reset } = useForm<FormData>({
    defaultValues: {
      name: "",
      email: "",
      message: "",
    },
  });

  useEffect(() => {
    async function fetchInitialData() {
      const res = await fetch("/api/interesting-data");
      const data: FormData = await res.json();
      reset(data); // prefill controlled fields
    }
    fetchInitialData();
  }, [reset]);

  const onSubmit = (data: FormData) => {
    console.log("Submitting", data);
    // Send to API...
  };

  return (
    <form onSubmit={handleSubmit(onSubmit)} noValidate>
      <Controller
        name="name"
        control={control}
        render={({ field }) => (
          <input {...field} placeholder="Name" />
        )}
      />
      <Controller
        name="email"
        control={control}
        render={({ field }) => (
          <input type="email" {...field} placeholder="Email" />
        )}
      />
      <Controller
        name="message"
        control={control}
        render={({ field }) => (
          <textarea {...field} placeholder="Message" />
        )}
      />
      <button type="submit">Submit</button>
    </form>
  );
}
```
