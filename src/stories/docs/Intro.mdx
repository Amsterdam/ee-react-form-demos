# Gemeente Amsterdam Form Demos

> **Warning**
> This project is a work in progress so will likely change in the future.

This project combines best practices, demos and example implementations of forms based on Gemeente Amsterdam projects and internal discussions.

It's intended to help (frontend) developers and designers choose the right aproach for their needs and build functional forms that are also accessible and maintainable.

For each demo we also try to include documentation with the following structure:

- General description
- Background/useful to know caveats
- Overview of the files required
- And the code from each file

## Our setup

Frontend applications in Gemeente Amsterdam use the component library [React](https://react.dev/). We've standardised two form options - pick the one that best fits your project:

- **HTML5** for quick, simple forms
- **React Hook Form** when you need advanced validation, performance and flexibility.
