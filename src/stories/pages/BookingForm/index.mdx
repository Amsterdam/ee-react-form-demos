## Description

This form is an example of a generic booking form. The key validation is with checking that the End date and End time fields are not older than the Start date and Start timefields.

This form's code, particularly the error state handling demonstrates how native HTML5 validation can quickly become cumbersome when you have a rule that dynamically targets more than one field.
